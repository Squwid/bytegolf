About

Byte Golf is a code solving problem game that awards players with the shortest code that solves the task at hand, just like golf! Inspired by Stack Overflow's Code Golf game open to the public.

Basic Rules

1. The player with the lowest total characters at the end wins, just like Golf!

2. Total score consists of characters or "bytes" in a working code solution (whitespace not included). The scores are totaled at the end, and all holes must be completed to a working solution to be able to win. Hence the name "Byte Golf"

3. Golfing in any order is allowed, as long as all hole solutions are completed by the conclusion of the game.

4. All code must be written in one of the supported languages and must only use standard libraries. Also must be in a single file for submission.

Open Source

This project is entirely open source, and I welcome all contributions that I can get.

Supported Languages

There are currently 14 supported languages.

Java, C, C++, C++14, PHP, Python 2, Python 3, Ruby, Go, Bash, Swift, R, NodeJS, F#.

Each submission must be in a single file and only contain the standard library.
